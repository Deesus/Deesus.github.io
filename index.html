About

Hi, my name is Dee Reddy. I'm a self-taught developer, currently residing in the Virginia/Maryland area. I'm a voracious learner -- whether that's reading about algorithm design or ancient philosophy, I am always looking to expand my understanding.

My primary languages are Python and JavaScript. I develop on both front-end and server-side. I develop web apps with either Node/Express or the Python-based Flask framework. I'm also experienced in SQL and RDBMS.

I develop responsive designs that work on all devices by using the latest HTML5 and CSS standards. As such, I prefer to find front-end solutions with CSS before relying on JS/jQuery. My approach is: plan, define, design, refine, and then deploy.
