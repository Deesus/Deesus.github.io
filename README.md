#### Single-page Profile Site

My Personal _GitHub Pages_ site. Feel free to fork and modify it to your liking.

##### Features:

- Responsive design
- Minimal
- Easily customizable
- Includes _home_, _about_, _portfolio_, and _contact_ sections
- JQuery enchanced
- Paralex scrolling
